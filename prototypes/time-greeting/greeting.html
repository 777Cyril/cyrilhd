Time-of-Day Greeting Prototype

The greeting appears on page load with typewriter effect, then fades away. It only shows once per 30-minute window.

How it works:

  • On page load, checks if greeting was shown in last 30 minutes
  • If not, displays appropriate greeting with typewriter effect
  • Greeting stays for ~3 seconds, then fades out
  • Uses sessionStorage + timestamp to track display timing
  • Quick refreshes won't retrigger the greeting
