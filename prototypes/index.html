Quietly Interesting Prototypes

Interactive features that add delight without distraction

Design Philosophy: These prototypes explore subtle, discoverable interactions that enhance the website experience while maintaining its minimalist aesthetic. Each feature is "quietly interesting" — something you notice and appreciate rather than being hit over the head with.

Song Carousel
Displays "you are now listening to [song name]" in a contained carousel with a sunset-style animation (slides from right to left). Positioned below the avatar and before the "hi" greeting.
  • Constrained width (adjustable 300-600px)
  • Smooth horizontal animation
  • Integrates with existing music player
  • Dark mode support
✓ Ready to Test
Time-of-Day Greeting
Shows a personalized greeting ("good morning", "good afternoon", "good evening", or "burning the midnight oil") on page load using the existing typewriter effect. Only appears once per 30-minute session to avoid repetition on quick refreshes.
  • Typewriter animation (matches music player style)
  • Smart session management (30min cooldown)
  • Time-based greeting selection
  • Auto-fades after display
✓ Ready to Test
Link Preview Whispers
Tiny preview cards that appear when hovering over links. Shows a title, description, and type (project/work/article/internal) with a 500ms delay to avoid being intrusive. Slides in from the side with minimal design.
  • 500ms hover delay (configurable)
  • Smart positioning (stays in viewport)
  • Uses data attributes for content
  • Lightweight and non-intrusive
✓ Ready to Test
Hidden Keystroke Patterns
Easter eggs triggered by typing "777" or "brr" anywhere on the page. Multiple animation options for each pattern including falling numbers, text animations, avatar spins, and page effects.
  • Detects "777" and "brr" patterns
  • Multiple animation styles per pattern
  • Cooldown to prevent spam
  • Doesn't interfere with form inputs
✓ Ready to Test
